HOW TO DEPLOY THE BLOG APP:
      ____PART A___
a) Assuming you are using Visual Studio with in Linux OS,
Run the following in your terminal;
$ pipenv shell 
      ____PART B___
Within your created virtual environment;
i) Download the requirements.
ii) Create super user. 
iii) python manage.py runserver
iv) Open up the admin portal i.e http://127.0.0.1:8000/admin/ assuming 8000 is your port
v) Create users, create your first post and publish.
vi) From your web run ie http://127.0.0.1:8000/ assuming 8000 is your port
or http://127.0.0.1:3000 assuming 3000 is your port number is 3000 to gain 
access to your created posts. 
vii) For extra help contact developers via LinkedIn;


Good luck!!!
